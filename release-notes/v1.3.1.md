# Wallpaper Player v1.3.1

这是 v1.3.0 的 Windows 热补丁版本，重点修复插件安装入口在 Windows 文件选择器里看不到 zip 插件包的问题。

## 修复

- 拆分插件安装入口：
  - “安装包”只打开文件选择器，支持选择 `.zip` 插件包或 `plugin.json`。
  - “安装文件夹”只打开目录选择器，用于选择插件根目录。
- 修复 Windows / Electron 在同一个对话框里同时启用文件和目录选择时，zip 文件被隐藏、无法选择的问题。

## 说明

- 插件包仍在 `release/plugins/` 下单独分发。
- 本次没有改变插件 manifest、生命周期、权限或视频理解能力。
- 如果你要安装 zip 插件包，请在“设置 -> 插件管理”中点击“安装包”。

## 验证

```powershell
npm run build
npm run verify:plugin-package
npm run verify:plugins
git diff --check
```
