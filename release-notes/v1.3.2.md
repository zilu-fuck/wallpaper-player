# Wallpaper Player v1.3.2

这是 v1.3.1 的 Windows 补丁版本，重点完善插件化后的稳定性、视频理解插件配置说明，以及安装版在不同机器上的路径迁移。

## 改进

- 视频列表增加分页，一页约 100 个视频，降低大库浏览时的渲染压力。
- 优化视频播放相关内存占用和卡顿表现，减少播放单个视频后资源占用突然升高的问题。
- 插件管理页和设置页补充亮色主题颜色修复，避免部分文字在浅色背景上看不清。
- 视频理解插件设置改为更直白的说明：
  - 普通用户只需要确认目录、准备视觉模型、点击“启动模型”。
  - 明确分析结果目录、模型目录、本地 VLM、外接 VLM API、文本模型各自用途。
  - 高级配置标注“通常不用改”，降低误改服务程序路径和启动参数的概率。

## 修复

- 修复安装版中视频理解“启动模型”按钮可能点不动的问题。
- 修复旧安装路径或相对路径导致 `llama-server.exe` 找不到的问题。
- 修复换电脑、重装或迁移配置后，旧机器保存的默认模型目录和结果目录仍被使用的问题。
- 如果旧机器保存的 VLM 模型绝对路径失效，新机器当前模型目录存在同名模型，会自动迁移到当前机器路径。
- 修复分析日志内容过长时超出卡片的问题。
- 修复全屏播放和部分主题颜色显示异常。

## 插件包

- 主程序和官方插件仍然分开打包。
- 插件包位于 `release/plugins/`：
  - `Wallpaper-Player-Plugin-video-analysis-1.0.0.zip`
  - `Wallpaper-Player-Plugin-ai-search-0.1.0.zip`
  - `Wallpaper-Player-Plugin-agent-bridge-0.1.0.zip`
- 视频理解插件仍需要在“设置 -> 插件管理”中安装并启用。
