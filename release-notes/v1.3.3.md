# Wallpaper Player v1.3.3

这是 v1.3.2 的便携版专项修复版本，重点解决便携版数据不持久化、手机端扫码绑定后连不上的问题。

## 修复

### 便携版数据持久化

- 修复便携版（单文件 exe）数据未写入 exe 旁目录、反而写入系统 `%APPDATA%` 的问题。
  - 便携版现在会将所有数据（设置、扫描缓存、缩略图、身份文件、插件配置等）保存到 exe 同级 `Data/` 目录。
  - 换电脑、换目录运行便携版不再丢失设置和已绑定手机。
  - 这是导致"便携版连不上手机端"的根因：身份文件每次启动被重新生成，accessToken 和 machineSecret 全部变化，手机端旧 token 全部失效。

### 手机连接稳定性

- 修复二维码里电脑地址可能选到虚拟网卡 IP（VMware / Hyper-V / WSL / Docker / VirtualBox 等）导致手机扫不上、连不上的问题。
  - 扫码绑定二维码生成时过滤虚拟网卡地址，优先使用物理网卡局域网 IP。
- 修复设备身份文件损坏时静默重新生成、用户无感知的问题。
  - 身份文件损坏或缺失关键字段时记录日志，并在"设置 -> 手机访问"页面顶部显示告警条，提示需要重新扫码绑定。
- 对齐手机端等待电脑端允许绑定的超时时间，从 2 分钟延长到 5 分钟，与电脑端二维码有效期一致。
  - 修复用户在电脑端没及时点"允许"时，手机端提前报"等待超时"的问题。

