## 更新内容

- 电脑端版本升级到 `1.2.3`，手机端 APK 版本升级到 `0.1.2`。
- 新增视频理解功能，可在电脑端配置本地或外接文本模型、视觉模型，并保存每个视频的最新分析结果。
- 新增视频分析侧边栏，支持查看分析队列、已有结果、结果弹窗、多选删除和单条删除。
- 视频播放器新增分析结果展示区域，分析结果会以“视频名 + 分析结果 + 识别码”的稳定文件名保存，新的分析会覆盖同一视频旧结果。
- 设置页新增视频理解模型目录、结果目录、VLM 下载/检测/启动配置入口，并优化了配置布局。
- 手机端新增视频分析入口，可从手机请求电脑端执行分析，并在手机端查看进度、摘要、标签和时间线结果。
- 关闭电脑端项目时会同步关闭由项目启动的 VLM 模型服务。

## 安装包

- Windows 安装包：`Wallpaper-Player-Setup-1.2.3.exe`
- Windows 便携版：`Wallpaper-Player-1.2.3.exe`
- Android APK：`wallpaper-player-mobile-0.1.2-arm64-release.apk`

## 验证

- `npm run build`
- `npm run verify:mobile-lan`
- `cd mobile && npm run typecheck`

## 已知限制

- Android APK 当前使用 debug keystore 签名的 release 构建，只用于内部测试。
- 视频理解依赖用户配置的文本模型/VLM 服务；本地模型不可用时，需要先在电脑端设置页完成模型下载、检测和启动。
- 自动更新依赖 GitHub Releases 网络可达性；无法访问 GitHub 时建议手动分发安装包和 APK。
