## 更新内容

- 电脑端版本升级到 `1.2.2`，手机端 APK 版本升级到 `0.1.1`，作为同一批局域网播放测试版发布。
- 电脑端新增多选视频批量添加标签入口，可对筛选结果中的多个视频一次性添加已有标签或新标签。
- 手机端支持长按视频进入多选，并在批量标签弹窗中搜索已有标签、选择已有标签和添加新标签。
- 修复手机端添加标签后电脑端不刷新的问题，远程标签变更会同步触发桌面端视频库和设置状态更新。
- 修复旧版标签数据使用完整路径作为键时导致标签不同步的问题，读写时会兼容并迁移到当前视频 ID。
- 优化绑定二维码内容，包含更多可用局域网地址，降低 VPN、多网卡环境下扫到不可达地址的概率。
- 优化手机端批量标签弹窗的键盘避让和确认区域，减少输入法遮挡。

## 安装包

- Windows 安装包：`Wallpaper-Player-Setup-1.2.2.exe`
- Windows 便携版：`Wallpaper-Player-1.2.2.exe`
- Android APK：`wallpaper-player-mobile-0.1.1-arm64-release.apk`

## 验证

- `npm run build`
- `npm run verify:remote-library`
- `npm run verify:mobile-lan`
- `cd mobile && npm run typecheck`

## 已知限制

- Android APK 当前使用 debug keystore 签名的 release 构建，只用于内部测试。
- 手机端直播放仍依赖系统原生解码能力，MP4/H.264/AAC 最稳定。
- 自动更新依赖 GitHub Releases 网络可达性；无法访问 GitHub 时建议手动分发安装包和 APK。
