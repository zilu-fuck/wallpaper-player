## 版本定位

`v1.2.4` 是 `v1.2.3` 视频理解版本的 Windows 补发版本，主要修复 GitHub Actions 上 `Release Windows` 工作流无法完成打包的问题。

## 更新内容

- 修复 `Release Windows` 工作流失败：CI 环境中没有本地被 `.gitignore` 排除的 `video comprehension` 子项目，导致 `scripts/package-win.js` 在打包阶段报错。
- 新增已纳入版本管理的最小视频理解运行时副本：`main/video-comprehension-runtime`，让 CI 可以稳定打出包含视频理解运行文件的安装包。
- 打包脚本优先使用本机 `video comprehension/video comprehension`，缺失时自动退回到 `main/video-comprehension-runtime`。
- 继续只携带运行所需文件，不把模型、输出、虚拟环境、缓存、本地 `.env` 等机器状态打进安装包。
- 版本号提升到标准 semver `1.2.4`，确保自动更新可以从 `1.2.3` 正常前进。
- 手机端本次不补发，Android APK 继续使用 `0.1.2` 版本。

## 安装包

- Windows 安装包：`Wallpaper-Player-Setup-1.2.4.exe`
- Windows 便携版：`Wallpaper-Player-1.2.4.exe`
- Android APK：继续使用 `wallpaper-player-mobile-0.1.2-arm64-release.apk`
