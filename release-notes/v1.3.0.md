## 版本定位

`v1.3.0` 是插件化和性能整理版本。主程序继续保持本地视频浏览、搜索、播放、手机访问等核心能力；视频理解、AI 搜索、Agent Bridge 改为插件形态分发和管理。

## 更新内容

- 新增插件注册表：主进程现在可统一注册插件贡献的 IPC、远程路由、设置 schema、能力和生命周期。
- 新增插件管理页：设置侧边栏可切换基础设置、手机访问、插件管理和系统状态；插件可安装、启用、停用、卸载和保存配置。
- 视频理解改为官方插件：视频分析 IPC、远程路由、VLM 服务和分析结果读取由 `video-analysis` 插件注册，核心主进程不再直接散落视频理解实现。
- 主程序和插件分开打包：Windows 主程序包不再内置视频理解实现和运行时；`release/plugins/` 会单独输出视频理解、AI 搜索、Agent Bridge 插件 zip。
- 支持直接安装插件 zip：插件管理页可选择插件目录、`plugin.json` 或 `.zip` 插件包。
- 新增 AI 搜索和 Agent Bridge 插件占位：两者按统一注册表进入插件列表，后续会通过受控能力访问视频库、视频理解摘要和播放器控制。
- 优化视频库内存和卡顿：缩略图和元数据改为可见区域懒加载，播放时暂停普通缩略图队列；列表分页默认每页 100 个视频。
- 优化桌面播放器和卡片菜单：修复全屏下菜单定位、亮色主题菜单颜色和移动端分析插件未启用提示。
- 修复桌面端视频理解可用性误判：现在以插件注册表的真实启用状态为准，避免插件未安装或激活失败时调用未注册 IPC。

## 注意事项

- 视频理解已从主程序拆成插件，升级后需要在“设置 -> 插件管理”安装并启用 `video-analysis` 插件包。
- AI 搜索和 Agent Bridge 目前是插件占位和受控能力设计，不开放任意文件或进程能力。
- 视频理解插件包含运行时代码和本地 VLM 服务程序，但不包含大模型文件；模型仍需用户按需配置或下载。
- Android APK 仍使用 debug keystore 签名的 release 构建，仅适合内部测试。
